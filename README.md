# NAME

App::sayhi

# VERSION

version 0.002

# SYNOPSIS

From the command line:

    sayhi          # Prints "Hello, World!" followed by a newline
    sayhi --shout  # Shouts it

# NAME

App::sayhi - backend interface to the \`sayhi\` command
